解法1:

拓扑排序

基本上照着拓扑排序模板来就行了
就是题意稍微理解一下
prerequisites[i] = [ai, bi] ，表示如果要学习课程 ai 则 必须 先学习课程 bi 。
所以 b -> a

拓扑排序步骤:

  1. 入队所有入度为0 的点
  2. bfs 取出队头t -> 每取出一次队头说明有一个数已加入排序好的序列
  3. 枚举t的所有出边 t->j
    1. 删掉 t->j 即让 d[j]– (d[j] j的入度)
    2. if d[j] == 0 -> 说明j之前所有点均以排好序 -> j 入队
  4. 如果图中存在环则无法全部入队 反之则都可以入队
